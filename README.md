# Trouve Ton Artisan

## Description
Projet réalisé dans le cadre de ma formation 
L’objectif est de créer un site permettant de rechercher un artisan à partir d’un mot‑clé, d’afficher les résultats et de consulter la fiche détaillée d’un artisan.

## Fonctionnalités du projet
- Barre de recherche (header + page résultats)
- Affichage des artisans sous forme de cartes
- Page de résultats avec filtres
- Page fiche artisan
- Design responsive (Desktop, Tablette, Mobile)

## Technologies utilisées
- Angular 21.1
- TypeScript
- HTML / SCSS
- Bootstrap
- Figma (maquettes)
- VS Code (développement)
